# chatBot

Creates .py files for self learning based chatBot

"chatbot with gui.py" python script is the main file to be run to interact with the chatbot.

"arithmetic.py" provides arithmetic capabilities of addition, subtraction, division and multiplication to the chatbot.

"general_ques.py" provides answer to the greetings and general questions like place of creation, gender, age etc.

"training data.txt" is a simple text file which contains text only data for chatbot based on the topic.

NOTE: If a user wants to run this chatbot first he/she needs to put all these four files in one directory then run the "chatbot with gui.py" script.
